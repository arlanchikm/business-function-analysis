# Анализ бизнес-функции для сервиса туроператора
В репозитории размещено описание бизнес-функции «Бронирование отеля» для сервиса туроператора.

## Описываемая функция
Бронирование отеля пользователем в онлайн-сервисе туроператора.

## Состав репозитория
- `hotel-booking-analysis.md` — основной документ с описанием бизнес-функции.
- `Hotel booking.bpmn` - BPMN-диаграмма бизнес-функции. 
- `Hotel booking_with_overlays.png` - PNG-экспорт BPMN-диаграммы.
- `hotel-booking-erd.png` — ERD логического уровня для функции бронирования отеля.

- описание границ функции
- конкурентный анализ
- бизнес-цели
- User Story

- ограничения
